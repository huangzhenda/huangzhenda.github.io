如何免费的搭建个人的博客

技术的选择

查找了下，目前搭建博客，主要用到以下几种方式：

  1. WordPress : 是一个以PHP和MySQL为平台的自由开源的博客软件和内容管理系统
  2. Jekyll : 简单静态博客网站生成器， 是用ruby写的
  3. octopress
  4. hexo

WordPress 太重， octopress 已经不怎么维护了， Jekyll 用的是ruby写的， hexo 用的是js。 相对于ruby，js会更熟悉点。最终选择 hexo来搭建博客

基于hexo搭建的博客

hexo是一个快速，简洁且高效的博客框架， 用Node.js写的，支持Markdown，部署安装都很简单，且拥有丰富的插件.

官方地址：https://hexo.io/zh-cn/

需要用到的工具

  1. Git
  2. Node.js
  3. GitHub Pages

Hexo的安装，部署，发布博文等

具体操作可以直接到官网上的文档上查询，步骤都写的很详细。

多台设备同步

可以用百度云类似的云存储，自动备份更新。只要在其他机子上把环境弄好，然后就可以同步写作了。

NEXT主题

https://github.com/iissnan/hexo-theme-next

遇到的问题

安装NodeJS， 无法找打npm。

如果在尝试安装 Hexo 的过程中出现 EACCES 权限错误，请遵循 由 npmjs 发布的指导 修复该问题。强烈建议 不要 使用 root、sudo 等方法覆盖权限

图片不显示问题

https://hexo.io/zh-cn/docs/asset-folders

  • 通过将 config.yml 文件中的 post_asset_folder 选项设为 true 来打开。随后，hexo new [layout] <title> 创建新文章的时候，会自动创建同名文件夹
  • 相对路径引用的标签插件。 随着Hexo 3 的发布，许多新的标签插件被加入到了核心代码中。这使得你可以更简单地在文章中引用你的资源。

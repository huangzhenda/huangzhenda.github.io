java学习笔记-基础

标识符与关键字

  • 标识符定义：
    • 字母、数字、_、$
    • 以英文字母开头。
  • 关键字： 系统对一些结构的描述处理，有特殊含义。

数据类型

  • 基础类型
    • 整型： byte 、 short 、 int 、long -> 默认值 0
    • 浮点型： float 、double -> 默认值：0.0
    • 布尔型： boolean -> 默认值： false
    • 字符型： char -> 默认值： \u0000
  • 引用类型
    • 数组、类型、接口 -> 默认值： null

数据范围！！！！

  • 如何处理超出的数据范围？ -> 如果超过最大的保存范围，会进行循环。 在java中成为数据溢出。
  • 解决数据溢出：
    • 在操作的时候预估数据范围，如果发现范围不够就使用更大的范围。
    • 可以直接追击 L 或者 long的转型。
  • 所有的数据类型进行自动转型的时候都是由小类型向大类型进行自动转换处理的。
  • 浮点型，默认是double。float需要添加 f/F。 float不精确。

运算

  • 数字运算
  • 条件运算
  • 逻辑运算
  • 位运算 ： 可以直接进行二进制数据的计算处理。
    • 与&、或｜、异或^、反码～、移位>> <<

逻辑控制

  • 条件 if
  • switch开关
  • 循环
    • while
    • do …… while
    • for
